O que é Economia Doméstica?

A Economia Doméstica é o conjunto de práticas e hábitos voltados à boa administração dos recursos dentro de um lar. Envolve planejar gastos, priorizar necessidades, evitar desperdícios e buscar equilíbrio entre renda e despesas.

Gerir bem a economia doméstica permite que as famílias tenham mais controle financeiro e consigam alcançar objetivos como a compra da casa própria, viagens ou a formação de uma reserva de emergência.

Finanças Pessoais: Dicas Essenciais

  • 1. Monte um orçamento mensal: anote todas as receitas e despesas para entender para onde vai seu dinheiro.
  • 2. Defina prioridades: diferencie o que é essencial (como alimentação e moradia) do que é supérfluo.
  • 3. Evite dívidas desnecessárias: use crédito de forma consciente e evite compras por impulso.
  • 4. Crie uma reserva de emergência: o ideal é guardar o equivalente a 3 a 6 meses das suas despesas fixas.
  • 5. Invista no seu futuro: aprenda sobre investimentos seguros, como Tesouro Direto, CDBs e fundos de investimento.

Comparação entre Plataformas de Hospedagem de Sites Estáticos

Hoje existem diversas plataformas que permitem hospedar sites estáticos gratuitamente e de forma simples. Abaixo, uma comparação entre as três mais populares: Vercel, GitHub Pages e Netlify.

Plataforma Vantagens Desvantagens
Vercel
  • Integração direta com o GitHub;
  • Deploy automático a cada commit;
  • Suporte excelente para frameworks como Next.js e React;
  • Interface moderna e fácil de usar.
  • Menos indicada para sites muito grandes ou com backend complexo.
GitHub Pages
  • Totalmente gratuito e sem necessidade de configurações complexas;
  • Ideal para portfólios, currículos e páginas simples;
  • Ligado diretamente ao seu repositório GitHub.
  • Funcionalidades limitadas;
  • Deploy manual em alguns casos;
  • Sem suporte nativo para backend.
Netlify
  • Deploy simples e rápido via Git;
  • Oferece funções serverless e formulários integrados;
  • Permite customização de domínio e SSL gratuito.
  • Algumas funções avançadas são pagas;
  • Pode ser mais complexo para iniciantes.

Conclusão

A publicação de um site estático é uma etapa importante no aprendizado do desenvolvimento web. Nesta atividade, além de revisar conceitos de **HTML e CSS**, foi possível entender como funciona o processo de deploy em plataformas modernas e gratuitas.

A Vercel se destaca pela praticidade e integração com o GitHub, sendo uma excelente escolha para quem está começando ou quer automatizar a publicação de projetos.
